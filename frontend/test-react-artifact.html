React Artifact Direct Test

Тестирование React артефакта напрямую

Метод 1: Через ссылку (artifact: protocol)

Кликни на ссылку для открытия артефакта:

[hello-react](artifact:hello-react) - React Test Component

Метод 2: Напрямую в iframe (для быстрого теста)

Если ссылка не работает, можно открыть основное приложение и найти урок:

  • Открой http://localhost:5174
  • Найди курс "react-artifacts-test" в сайдбаре (может быть внизу списка)
  • Открой урок "React Artifacts Testing"
  • Кликни на ссылку [hello-react](artifact:hello-react)

Ожидаемое поведение:

  1. Loading состояние (spinner + "Loading component...")
  2. Затем загрузится компонент с градиентом indigo-purple
  3. Заголовок: "React Artifacts Test"
  4. Сообщение: "React.lazy + Suspense + Error Boundary working correctly!"
  5. Список с галочками ✅
  6. Timestamp загрузки

Отладка:

Открой DevTools (F12) → Console для просмотра ошибок
