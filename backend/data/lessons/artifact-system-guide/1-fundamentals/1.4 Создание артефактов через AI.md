# Создание артефактов через AI

🚧 **Статус:** В разработке (Фаза 4 roadmap)

В этом уроке вы узнаете, как AI будет генерировать артефакты во время диалога. Это будущая функциональность, которая сейчас находится в процессе реализации.

---

## Как это будет работать?

### Сценарий использования

1. **Вы спрашиваете AI:**
   - "Создай график температуры за неделю"
   - "Построй столбчатую диаграмму продаж"
   - "Сделай калькулятор площади круга"

2. **AI отвечает:**
   - Короткое объяснение (1-2 предложения)
   - JSON конфигурация артефакта
   - Автоматически открывает артефакт

3. **Артефакт открывается:**
   - В CENTER контейнере (fullscreen)
   - Кнопка `📄 Show Lesson` показывает урок рядом
   - Можно сразу работать с артефактом

---

## Примеры промптов

### Для графиков

**✅ Хорошие промпты:**

```
"Создай график температуры по дням недели:
Пн:18°C, Вт:20°C, Ср:19°C, Чт:22°C, Пт:24°C, Сб:23°C, Вс:21°C"

"Построй линейный график роста пользователей:
Янв:100, Фев:150, Мар:230, Апр:350, Май:520, Июн:780"

"Покажи столбчатую диаграмму сравнения продаж:
Q1: Product A=45, B=60, C=38
Q2: Product A=52, B=55, C=47"

"Создай график синусоиды от 0 до 2π"

"Построй multi-line график: CPU, RAM, Disk за 24 часа"
```

**Почему хорошие:**
- ✅ Чётко указан тип графика
- ✅ Данные приведены явно
- ✅ Понятно, что на осях X и Y
- ✅ Есть контекст (что за данные)

**❌ Плохие промпты:**

```
"Нарисуй график"
"Покажи данные"
"Сделай визуализацию"
```

**Почему плохие:**
- ❌ Не указан тип графика
- ❌ Нет данных
- ❌ Непонятно, что визуализировать

---

### Для калькуляторов

**✅ Хорошие промпты:**

```
"Создай калькулятор площади круга:
Вход: радиус (1-100 см)
Выход: площадь, длина окружности"

"Сделай калькулятор скорости:
Вход: расстояние (км), время (часы)
Выход: скорость (км/ч), скорость (м/с)"

"Построй конвертер температуры:
Вход: Celsius
Выход: Fahrenheit, Kelvin"

"Создай калькулятор кинетической энергии:
Вход: масса (kg), скорость (m/s)
Выход: энергия (J)"
```

**Почему хорошие:**
- ✅ Указаны inputs (входные параметры)
- ✅ Указаны outputs (результаты)
- ✅ Даны единицы измерения
- ✅ Понятен диапазон значений

**❌ Плохие промпты:**

```
"Сделай калькулятор"
"Посчитай формулу"
"Нужен расчёт"
```

**Почему плохие:**
- ❌ Не указано, что считать
- ❌ Нет параметров
- ❌ Нет формулы

## Формат ответа AI

### Для графиков (Plotly)

AI будет отвечать в таком формате:

```
I'll create a temperature chart for you.

{
  "type": "plot",
  "title": "Weekly Temperature",
  "config": {
    "title": "Temperature Over Week",
    "data": [{
      "x": ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"],
      "y": [18, 20, 19, 22, 24, 23, 21],
      "type": "scatter",
      "mode": "lines+markers",
      "name": "Temperature (°C)",
      "line": { "color": "#f59e0b", "width": 3 },
      "marker": { "size": 8, "color": "#f59e0b" }
    }],
    "layout": {
      "title": "Weekly Temperature",
      "xaxis": { "title": "Day of Week" },
      "yaxis": { "title": "Temperature (°C)" }
    }
  }
}

[Artifact will open automatically]
```

**Формат JSON:**
- `type`: `"plot"` (тип артефакта)
- `title`: Заголовок для toolbar
- `config`: Plotly конфигурация
  - `data`: массив серий
  - `layout`: настройки осей, заголовки

---

### Для калькуляторов (Math.js)

AI будет отвечать в таком формате:

```
I'll create a circle area calculator.

{
  "type": "calculator",
  "title": "Circle Area Calculator",
  "config": {
    "title": "Circle Calculator",
    "inputs": [
      {
        "name": "radius",
        "label": "Radius",
        "type": "number",
        "min": 1,
        "max": 100,
        "step": 0.5,
        "default": 10,
        "unit": "cm"
      }
    ],
    "formulas": [
      {
        "name": "area",
        "label": "Area",
        "expression": "PI * radius^2",
        "unit": "cm²",
        "precision": 2,
        "showFormula": true
      },
      {
        "name": "circumference",
        "label": "Circumference",
        "expression": "2 * PI * radius",
        "unit": "cm",
        "precision": 2,
        "showFormula": true
      }
    ]
  }
}

[Artifact will open automatically]
```

**Формат JSON:**
- `type`: `"calculator"` (тип артефакта)
- `title`: Заголовок
- `config`: Math.js конфигурация
  - `inputs`: параметры ввода
  - `formulas`: формулы расчёта

## Правила для AI (system prompt)

AI будет обучен следующим правилам:

### Когда создавать артефакты

**✅ Создавать артефакт когда:**
- Пользователь явно просит график/калькулятор
- Запрос содержит данные для визуализации
- Нужно показать математический расчёт
- Пользователь спрашивает "покажи", "построй", "создай"

**❌ НЕ создавать артефакт когда:**
- Просто объясняем концепцию
- Данных для визуализации нет
- Вопрос теоретический
- Уже есть готовый шаблон (лучше дать ссылку)

### Выбор типа графика

AI будет выбирать тип на основе данных:

- **Line Chart:** Временные ряды, тренды, непрерывные данные
- **Scatter Plot:** Корреляции, распределения
- **Bar Chart:** Сравнение категорий, дискретные данные
- **Multi-Line:** Сравнение нескольких переменных
- **Area Chart:** Объём, накопительные данные

### Цвета и стили

AI будет использовать палитру:
- `#6366f1` - Indigo (основной)
- `#f59e0b` - Orange (акцент)
- `#10b981` - Green (позитив)
- `#ef4444` - Red (негатив/предупреждение)
- `#8b5cf6` - Purple (дополнительный)

## Технические детали (для понимания)

### Событие artifact:open

AI будет вызывать JavaScript событие:

```javascript
window.dispatchEvent(new CustomEvent('artifact:open', {
  detail: {
    type: 'plot',              // или 'calculator'
    title: 'Chart Title',
    config: { ... },           // Plotly или Math.js конфиг
    source: 'chat'             // Источник (для fullscreen режима)
  }
}));
```

**Обработчик события:**
- [CenterContainer.jsx:49-77](/frontend/src/components/center/CenterContainer.jsx#L49-L77)
- При `source: 'chat'` → открывается FULLSCREEN
- При `source: 'lesson'` → открывается SPLIT VIEW
