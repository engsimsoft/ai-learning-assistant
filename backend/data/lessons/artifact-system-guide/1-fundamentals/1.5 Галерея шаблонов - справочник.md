# Галерея шаблонов - справочник

Полный каталог всех доступных шаблонов артефактов с живыми примерами и описанием использования.

## 📊 Графики (Plots) - 5 шаблонов

### 1. Line Chart - Линейный график

**ID:** `line-chart`

**Назначение:** Непрерывные данные, тренды, временные ряды

**Примеры:**

- [📊 Базовый линейный график](artifact:line-chart)
- [📊 Температура за неделю](artifact:line-chart:temperature)
- [📊 Рост пользователей](artifact:line-chart:growth)

**Когда использовать:**
- Температура за период
- Рост показателей
- Динамика курсов
- Любые непрерывные данные

**Параметры:**
- `mode`: 'lines', 'markers', 'lines+markers'
- `line.color`, `line.width`, `line.dash`
- `marker.size`, `marker.symbol`

---

### 2. Scatter Plot - Точечная диаграмма

**ID:** `scatter-plot`

**Назначение:** Корреляции, распределение данных, паттерны

**Примеры:**

- [📊 Базовый scatter plot](artifact:scatter-plot)

**Когда использовать:**
- Корреляция рост/вес
- Распределение оценок
- Кластеризация данных
- Поиск выбросов

**Параметры:**
- `mode`: 'markers' (только точки)
- `marker.opacity`: прозрачность для плотных данных
- `marker.color`: может быть массивом (градиент)

---

### 3. Bar Chart - Столбчатая диаграмма

**ID:** `bar-chart`

**Назначение:** Сравнение категорий, дискретные данные

**Примеры:**

- [📊 Базовые столбцы](artifact:bar-chart)
- [📊 Продажи по месяцам](artifact:bar-chart:sales)
- [📊 Сравнение Q1 vs Q2 (grouped)](artifact:bar-chart:comparison)
- [📊 Трафик по устройствам (stacked)](artifact:bar-chart:stacked)
- [📊 Горизонтальные столбцы](artifact:bar-chart:horizontal)

**Когда использовать:**
- Сравнение продаж
- Показатели по категориям
- Результаты опросов
- Рейтинги и топы

**Параметры:**
- `orientation`: 'v' (вертикальные) или 'h' (горизонтальные)
- `barmode`: 'group' (рядом) или 'stack' (накопительно)
- `bargap`: промежуток между столбцами

---

### 4. Multi-Line Chart - Несколько линий

**ID:** `multi-line`

**Назначение:** Сравнение нескольких переменных одновременно

**Примеры:**

- [📊 Три серии данных](artifact:multi-line)
- [📊 Температура в 3 городах](artifact:multi-line:temperature)
- [📊 Метрики сервера (CPU, RAM, Disk)](artifact:multi-line:metrics)
- [📊 Рост по платформам](artifact:multi-line:growth)
- [📊 Финансы (доход, расход, прибыль)](artifact:multi-line:financial)

**Когда использовать:**
- Сравнение температур в городах
- Метрики системы
- Финансовые показатели
- Тренды в разных категориях

**Советы:**
- Максимум 5-6 линий (иначе сложно читать)
- Используйте контрастные цвета
- Включайте легенду

**Палитра:**
- `#6366f1` - Indigo
- `#f59e0b` - Orange
- `#10b981` - Green
- `#ef4444` - Red
- `#8b5cf6` - Purple

---

### 5. Area Chart - Площадной график

**ID:** `area-chart`

**Назначение:** Объём, накопительные данные, величина

**Примеры:**

- [📊 Базовый area chart](artifact:area-chart)
- [📊 Накопительные продажи](artifact:area-chart:sales)
- [📊 Трафик за 24 часа](artifact:area-chart:traffic)
- [📊 Стек устройств (stacked)](artifact:area-chart:stacked)
- [📊 Ресурсы сервера](artifact:area-chart:resources)
- [📊 Рост пользователей](artifact:area-chart:growth)

**Когда использовать:**
- Накопительные продажи
- Трафик сайта
- Использование ресурсов
- Объём производства

**Типы заливки:**
- `fill: 'tozeroy'` - до оси Y
- `fill: 'tonexty'` - до следующей серии (stacked)
- `fillcolor: 'rgba(R, G, B, A)'` - цвет с прозрачностью

## 🧮 Калькуляторы (Calculators) - 3 шаблона

### 1. Generic Calculator - Базовый калькулятор

**ID:** `generic-calculator`

**Назначение:** Простые вычисления с параметрами

**Примеры:**

- [🧮 Generic Calculator](artifact:generic-calculator)

**Когда использовать:**
- Любые простые расчёты
- Демонстрация Math.js
- Прототипирование формул

---

### 2. Unit Converter - Конвертер единиц

**ID:** `unit-converter`

**Назначение:** Преобразование между единицами измерения

**Примеры:**

- [🧮 Температура (C, F, K)](artifact:unit-converter)
- [🧮 Расстояние (m, km, mi, ft)](artifact:unit-converter:distance)
- [🧮 Скорость (km/h, m/s, mph, knots)](artifact:unit-converter:speed)
- [🧮 Вес (kg, lbs, oz)](artifact:unit-converter:weight)
- [🧮 Объём (L, gal, ml)](artifact:unit-converter:volume)
- [🧮 Двигатель: Рабочий объём (cc, ci, L)](artifact:unit-converter:engineDisplacement)

**Когда использовать:**
- Курсы с разными системами единиц
- Международная аудитория (метрическая vs имперская)
- Двигатели (cc/ci, kW/HP, Nm/lb-ft)

**Примеры для двигателей:**
```
Displacement: cc → ci, L
Power: kW ↔ HP
Torque: Nm ↔ lb-ft
Pressure: bar ↔ psi
```

---

### 3. Formula Calculator - Калькулятор формул

**ID:** `formula-calculator`

**Назначение:** Математические и физические формулы

**Примеры:**

- [🧮 Квадратное уравнение](artifact:formula-calculator)
- [🧮 Площадь круга](artifact:formula-calculator:circle)
- [🧮 Треугольник](artifact:formula-calculator:triangle)
- [🧮 Кинетическая энергия](artifact:formula-calculator:physics)
- [🧮 Сложные проценты](artifact:formula-calculator:finance)
- [🧮 Теорема Пифагора](artifact:formula-calculator:pythagoras)
- [🧮 BMI калькулятор](artifact:formula-calculator:bmi)

**Когда использовать:**
- Геометрия (площади, объёмы)
- Физика (энергия, мощность, сила)
- Финансы (проценты, амортизация)
- Инженерия (любые формулы)

**Примеры для двигателей:**
```
Compression Ratio: CR = (Vd + Vc) / Vc
Otto Efficiency: η = 1 - (1/CR)^(γ-1)
Engine Power: P = (T * RPM) / 9549
Piston Speed: Sp = 2 * Stroke * RPM / 60
```

## Для курса по двигателям

### Рекомендуемые шаблоны

**Графики:**

1. **Line Chart** - кривые крутящего момента/мощности
   ```markdown
   [📊 Torque Curve](artifact:line-chart:torque-curve)
   ```

2. **Multi-Line** - сравнение циклов
   ```markdown
   [📊 Otto vs Diesel Cycles](artifact:multi-line:cycles)
   ```

3. **Area Chart** - P-V диаграмма (заполненная область цикла)
   ```markdown
   [📊 P-V Diagram](artifact:area-chart:pv-diagram)
   ```

4. **Bar Chart** - сравнение эффективности
   ```markdown
   [📊 Efficiency Comparison](artifact:bar-chart:efficiency)
   ```

**Калькуляторы:**

1. **Compression Ratio** (степень сжатия)
   ```javascript
   inputs: [bore, stroke, clearance_volume]
   formulas: [displacement, compression_ratio]
   ```

2. **Otto Efficiency** (КПД цикла Отто)
   ```javascript
   inputs: [compression_ratio, gamma]
   formulas: [theoretical_efficiency, practical_efficiency]
   ```

3. **Engine Power** (мощность двигателя)
   ```javascript
   inputs: [torque_Nm, rpm]
   formulas: [power_kW, power_HP]
   ```

4. **Unit Converters**
   - cc → ci, L
   - kW ↔ HP
   - Nm ↔ lb-ft
   - bar ↔ psi

## Как создать свой шаблон

**Шаги:**

1. Скопируйте похожий существующий шаблон
2. Измените ID, name, description
3. Обновите config (data/inputs/formulas)
4. Добавьте examples (2-3 варианта)
5. Напишите instructions (как использовать)
6. Зарегистрируйте в `artifactTemplates.js`
7. Протестируйте!

**Файлы:**
- Графики: `frontend/src/templates/plots/my-template.js`
- Калькуляторы: `frontend/src/templates/calculators/my-template.js`
- Реестр: `frontend/src/templates/artifactTemplates.js`
