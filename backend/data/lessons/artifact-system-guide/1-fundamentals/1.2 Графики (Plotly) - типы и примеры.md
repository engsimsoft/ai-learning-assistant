# Графики (Plotly) - типы и примеры

Plotly.js — мощная библиотека для создания интерактивных графиков. В этом уроке вы увидите **все типы графиков**, которые поддерживает система артефактов.

---

## Возможности Plotly графиков

✅ **Zoom/Pan** - масштабирование колёсиком мыши, перемещение
✅ **Hover** - детали при наведении на точки
✅ **Export PNG** - сохранение как изображение
✅ **Reset view** - сброс масштаба
✅ **Toggle legend** - показать/скрыть легенду
✅ **Toggle series** - включить/выключить серии данных

---

## Типы графиков

### 1. Line Chart (Линейный график)

**Назначение:** Непрерывные данные, тренды, временные ряды

**Когда использовать:**
- Температура за период
- Рост показателей
- Динамика курсов
- Траектории движения

**Базовый пример:**

[📊 Line Chart - базовый](artifact:line-chart)

**Примеры с данными:**

[📊 Температура за неделю](artifact:line-chart:temperature)

[📊 Рост пользователей](artifact:line-chart:growth)

**Настройки:**
- `mode`: `'lines'`, `'markers'`, `'lines+markers'`
- `line.color`: цвет линии
- `line.width`: толщина (1-5)
- `line.dash`: `'solid'`, `'dash'`, `'dot'`
- `marker.size`: размер точек (4-12)

---

### 2. Scatter Plot (Точечная диаграмма)

**Назначение:** Корреляции, распределение данных, поиск паттернов

**Когда использовать:**
- Корреляция между переменными (рост/вес)
- Распределение оценок
- Кластеризация данных
- Анализ выбросов

**Пример:**

[📊 Scatter Plot - базовый](artifact:scatter-plot)

**Особенности:**
- Только точки (без линий)
- `mode: 'markers'`
- Можно использовать `opacity` для плотных данных
- Поддержка цветовых градиентов

---

### 3. Bar Chart (Столбчатая диаграмма)

**Назначение:** Сравнение категорий, дискретные данные

**Когда использовать:**
- Сравнение продаж по месяцам
- Показатели по категориям
- Результаты опросов
- Рейтинги и топы

**Базовый пример:**

[📊 Bar Chart - базовый](artifact:bar-chart)

**Варианты использования:**

[📊 Продажи по месяцам](artifact:bar-chart:sales)

[📊 Сравнение Q1 vs Q2 (grouped)](artifact:bar-chart:comparison)

[📊 Трафик по устройствам (stacked)](artifact:bar-chart:stacked)

[📊 Горизонтальные столбцы](artifact:bar-chart:horizontal)

**Настройки:**
- `orientation`: `'v'` (вертикальные) или `'h'` (горизонтальные)
- `barmode`: `'group'` (рядом) или `'stack'` (накопительно)
- `bargap`: промежуток между столбцами (0-1)
- `marker.color`: цвет столбцов

---

### 4. Multi-Line Chart (Несколько линий)

**Назначение:** Сравнение нескольких переменных одновременно

**Когда использовать:**
- Сравнение температур в разных городах
- Метрики сервера (CPU, RAM, Disk)
- Финансовые показатели (доход, расход, прибыль)
- Тренды в разных категориях

**Базовый пример:**

[📊 Multi-Line - 3 серии](artifact:multi-line)

**Примеры с данными:**

[📊 Температура в 3 городах](artifact:multi-line:temperature)

[📊 Метрики сервера (CPU, RAM, Disk)](artifact:multi-line:metrics)

[📊 Рост по платформам](artifact:multi-line:growth)

[📊 Финансы (доход, расход, прибыль)](artifact:multi-line:financial)

**Советы:**
- Используйте разные цвета для каждой линии
- Максимум 5-6 линий (иначе сложно читать)
- Включайте легенду для идентификации
- Выбирайте контрастные цвета

**Палитра цветов:**
- `#6366f1` - Indigo (синий)
- `#f59e0b` - Orange (оранжевый)
- `#10b981` - Green (зелёный)
- `#ef4444` - Red (красный)
- `#8b5cf6` - Purple (фиолетовый)

---

### 5. Area Chart (Площадной график)

**Назначение:** Показать объём, величину, накопительные данные

**Когда использовать:**
- Накопительные продажи
- Трафик сайта
- Использование ресурсов
- Объём производства

**Базовый пример:**

[📊 Area Chart - базовый](artifact:area-chart)

**Примеры:**

[📊 Накопительные продажи](artifact:area-chart:sales)

[📊 Трафик за 24 часа](artifact:area-chart:traffic)

[📊 Стек устройств (stacked)](artifact:area-chart:stacked)

[📊 Ресурсы сервера](artifact:area-chart:resources)

[📊 Рост пользователей](artifact:area-chart:growth)

**Типы заливки:**
- `fill: 'tozeroy'` - до оси Y (одиночный график)
- `fill: 'tonexty'` - до следующей серии (stacked)
- `fill: 'toself'` - замкнутая область

**Настройки прозрачности:**
```javascript
fillcolor: 'rgba(99, 102, 241, 0.3)'  // 30% прозрачности
fillcolor: 'rgba(16, 185, 129, 0.5)'  // 50% прозрачности
```

## Общие параметры графиков

### Оси (xaxis / yaxis)

```javascript
xaxis: {
  title: 'X Axis Label',    // Подпись оси
  gridcolor: '#e5e7eb',     // Цвет сетки
  showgrid: true,           // Показывать сетку
  range: [0, 100],          // Диапазон (optional)
  type: 'linear'            // 'linear', 'log', 'date', 'category'
}
```

### Заголовок и легенда

```javascript
layout: {
  title: 'Chart Title',
  showlegend: true,
  legend: {
    x: 1,              // Позиция (0-1)
    xanchor: 'right',  // Привязка
    y: 1
  }
}
```

### Цвета и темы

Текущая тема: **Светлая** (white background)
- `paper_bgcolor`: `'#ffffff'` - фон вокруг
- `plot_bgcolor`: `'#ffffff'` - фон графика
- Тёмная тема пока не поддерживается

## Сравнение типов графиков

| Тип | Лучше для | Плохо для |
|-----|-----------|-----------|
| **Line** | Тренды, временные ряды | Категории, дискретные данные |
| **Scatter** | Корреляции, распределения | Точные значения |
| **Bar** | Сравнение категорий | Непрерывные данные |
| **Multi-Line** | Сравнение трендов | Много серий (>6) |
| **Area** | Объём, накопительные | Точные значения |

## Практическое задание

### Задание 1: Изучите примеры

Откройте по очереди все графики выше и обратите внимание:

1. **Интерактивность:**
   - Наведите на точки (hover) - видите детали?
   - Прокрутите колёсиком (zoom) - масштабируется?
   - Нажмите `↺ Reset` - вернулось к исходному виду?

2. **Кнопка `</>`:**
   - Откройте код конфигурации
   - Изучите структуру `data` и `layout`
   - Найдите, где задан цвет линии

3. **Экспорт:**
   - Нажмите `📥 Export`
   - Сохранился PNG-файл?

### Задание 2: Для курса по двигателям

**Подумайте:** Какие графики понадобятся для курса по двигателям?

Возможные варианты:
- **Line Chart**: Кривая крутящего момента
- **Area Chart**: P-V диаграмма (заполненная область цикла)
- **Multi-Line**: Сравнение циклов (Отто, Дизель, Аткинсона)
- **Bar Chart**: Сравнение эффективности разных двигателей
